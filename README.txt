Embed content from Monkey in Silk projects in Wordpress.

== Description ==

This plugin allows you to easily embed content you have purchased from [Monkey in Silk](https://monkeyinsilk.se/). It includes shortcodes to include the following products:

* Homeselector
* Homestyler
* Object viewer
* Customizer
* Fake 3d

This plugin is added for convenience and is not necessary to embed content from Monkey in Silk, but it is recommended. 

== Frequently Asked Questions ==

= Do I need to purchase products from Monkey in Silk to benefit from using this plugin? =

Yes. The plugin can be used freely but if you don't have any content hosted by Monkey in Silk it can only be used to test intgration with our publicly available demo projects.
